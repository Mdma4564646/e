#рсз
Получите значения R,S,Z подписи ECDSA из блокчейна rawtx или txid.

## Информация
Скрипт анализирует данные rawtx, чтобы получить все входные данные транзакции, и восстанавливает неподписанное сообщение для каждого из них.
  чтобы найти значение Z. Результат предоставляется как R,S,Z,Pubkey для каждого из входных данных, присутствующих в данных rawtx. _**[Интернет не требуется]**_
 
  Если вместо rawtx указан txid, то API блокчейна используется для получения сведений о rawtx, а затем начинается расчет R,S,Z. _**[Требуется Интернет]**_

## Требования
Необходимую библиотеку (3 файла) можно получить по адресу https://github.com/iceland2k14/secp256k1.
